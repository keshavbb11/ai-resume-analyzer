# ai-resume-analyzer
AI-powered resume analyzer that evaluates resumes against job descriptions using GenAI, semantic search, and ATS-style scoring with actionable recommendations.

What this project does

 Accepts multiple resumes (PDF)
 Compares resumes against a job description
 Uses GenAI + semantic search (RAG) to understand content
 Generates role-based scores
 Ranks candidates automatically
 Suggests missing skills and improvements
 Generates downloadable PDF reports
 Removes personal details to reduce bias
 Runs fully locally (no paid APIs)
